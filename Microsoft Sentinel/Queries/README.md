# Sentinel Queries

KQL queries are organized by scenario so they are easier to browse, reuse, and reference from documentation.

| Folder | Description |
| --- | --- |
| `Identity/` | Queries for sign-in activity, travel anomalies, and authentication patterns. |
| `Network/` | Queries for domain and network connection analysis. |
| `Log-Ingestion/` | Queries for monitoring ingestion volume, spikes, and log source behavior. |
| `Endpoint/` | Queries for endpoint or VM telemetry availability. |
| `User-Behavior/` | Queries that use behavior analytics or anomaly scoring. |
