# Sentinel Automation

This folder contains Logic App and playbook templates related to Microsoft Sentinel operations.

| Folder | Description |
| --- | --- |
| `Logic-Apps/` | General Logic App workflows used with Sentinel or supporting Azure security operations. |
| `Playbooks/` | Sentinel playbook templates intended for incident response or monitoring workflows. |

Review and replace placeholder values such as subscription IDs, resource groups, recipients, and connections before deploying.
