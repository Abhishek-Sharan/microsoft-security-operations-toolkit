# Azure Monitor Agent Management

Scripts for Azure Monitor Agent and extension operations across Azure VMs and Azure Arc-enabled servers.

| Folder | Description |
| --- | --- |
| `Scripts/Azure-VMs/` | Install, uninstall, and bulk-manage AMA extensions for Azure VMs in a resource group. |
| `Scripts/Azure-Arc/` | Manage AMA and extension auto-upgrade for Azure Arc-enabled servers. |
| `Scripts/Linux/` | Validate and tune AMA cache behavior on Linux systems. |
| `Scripts/Reports/` | Generate extension-related operational reports. |

Review each script and test in a controlled environment before production use.
